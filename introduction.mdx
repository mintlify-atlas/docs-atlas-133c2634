# Bienvenido a TMT

TMT es un sistema completo de gestión de boletería construido sobre Firebase Cloud Functions. Proporciona una API robusta para la venta de boletos, procesamiento de pagos, control de acceso y facturación digital automatizada.

## Arquitectura del Sistema

TMT está construido como un conjunto de Firebase Cloud Functions que se comunican con:

- **Firestore**: Base de datos principal para eventos, boletos, órdenes y usuarios
- **PostgreSQL**: Base de datos secundaria para consultas analíticas y reportes
- **Firebase Auth**: Gestión de autenticación de usuarios
- **Firebase Storage**: Almacenamiento de archivos y documentos

<Note>
Todas las funciones están diseñadas para ser invocadas como HTTPS requests con CORS habilitado para integración desde múltiples clientes.
</Note>

## Tipos de Usuarios

El sistema soporta tres tipos principales de usuarios:

1. **Clientes** (`u_clients`): Organizadores de eventos que crean y administran sus propios eventos
2. **Colaboradores** (`u_collaborators`): Personal que trabaja para los clientes
3. **Staff** (`u_staff`): Personal administrativo de la plataforma TMT

<Info>
Cada tipo de usuario tiene permisos y capacidades diferentes dentro del sistema. Consulta la sección de [Autenticación](/auth/overview) para más detalles.
</Info>

## Flujo Principal

<Steps>
  <Step title="Crear un evento">
    Un cliente crea un evento con fechas, zonas y configuración de asientos
  </Step>
  <Step title="Generar boletos">
    El sistema genera automáticamente todos los boletos basándose en la configuración de zonas
  </Step>
  <Step title="Asignar taquillas">
    Se configuran taquillas (físicas u online) para vender los boletos
  </Step>
  <Step title="Procesar órdenes">
    Los compradores realizan órdenes que incluyen boletos y transacciones de pago
  </Step>
  <Step title="Control de acceso">
    En el día del evento, se validan los boletos y se registran entradas/salidas
  </Step>
  <Step title="Facturación y conciliación">
    Se generan facturas automáticamente y se concilian los pagos con los bancos
  </Step>
</Steps>
