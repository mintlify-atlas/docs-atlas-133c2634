## Overview

This endpoint retrieves all tickets associated with a specific event from the PostgreSQL database. It returns complete ticket information including status, seat assignments, and event details.

## Request

<ParamField body="data" type="object" required>
  Request data object
  
  <ParamField body="data.event_id" type="string" required>
    The unique identifier of the event whose tickets you want to retrieve
  </ParamField>
</ParamField>

## Notes

- Returns all tickets for the event regardless of their status (available, sold, or used)
- The response includes customer information if tickets have been purchased
- The ledger field provides a complete audit trail of all ticket actions
- Use `status: true` to identify available tickets
- Use `access_status` and `access_entry` to track real-time event attendance
